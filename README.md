# Jobifyy

## Project Description
Jobifyy is a powerful job search and application tracking platform designed to help users streamline their job hunt. With features that allow users to search for jobs, track applications, and manage their career prospects in one easy-to-use interface, Jobifyy empowers job seekers to take control of their job search.

## Features
- **Job Search:** Find job postings from various sources.
- **Application Tracking:** Keep track of job applications, interviews, and follow-ups.
- **User-friendly Dashboard:** A clean and intuitive interface for easy navigation.
- **Notifications:** Get alerted on job matches and application statuses.

## Tech Stack
- **Frontend:** React, Redux, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Heroku

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/bhartihere/Jobifyy.git
   cd Jobifyy
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables (create a `.env` file and add the necessary variables).
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
Navigate to `http://localhost:3000` in your web browser to start using Jobifyy. You can begin by searching for jobs or tracking your applications.
